# ChatVTX: Smart AI Hub (Llama-3-8B)

Готовый каркас для развертывания нейросетевого ассистента на базе модели **Llama-3-8B** с интегрированной панелью мониторинга.

## Основные возможности
- **Интеллект:** Оптимизировано под работу с моделью 8B (формат GGUF).
- **Admin Hub:** Встроенный мониторинг активности пользователей в реальном времени.
- **Безопасность:** Доступ к API защищен мастер-ключом.
- **Легкий вес:** Минимальные зависимости, быстрая установка на Vast.ai или любой GPU-сервер.

## Быстрый старт
1. Склонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Поместите модель Llama-3-8B в папку `models/`.
4. Запустите сервер через `./re.sh`.

## Технологии
- Python / Flask
- Llama-cpp (с поддержкой GPU ускорения)
- DuckDNS / SSH Tunneling
